Very Basic Concepts

You should be familiar with the very basics of how electricity works. There are a million books and websites to learn from. We suggest the following webpages, written by Sparkfun, a big vendor of electronics.

  1. What is Electricity?
  2. Electric Power
  3. Voltage, Current, Resistance, and Ohm’s Law
  4. What is a Circuit?
  5. How to Use a Multimeter
  6. Voltage Dividers
  7. How to Use a Breadboard

There is no end to learning about these topics! The good news is that you can accomplish a lot of stuff with the very basic notions of electric circuits. If you want to learn even more, navigate your way on the “Suggested Reading” section in the tutorials above.
